群晖装entware&aria2

发表于 2019-10-19 | 分类于 n1 | | 阅读次数:

安装entware

  • 首先在群晖的套件中心添加第三方源：https://www.cphub.net
  • 然后在套件中心的社群中，安装Easy Bootstrap Installer。
  • 安装的时候选择entware，软件更多

安装aria2

  • ssh登录切换到root权限

  • 配置文件在 /opt/etc/aria2.conf

安装 AriaNg

  • 首先到 https://github.com/mayswind/AriaNg/releases 下载好
  • 群晖套件中心配置好 web station
  • 把文件copy到web文件夹即可
