快速幂取模

有时候幂运算所得到结果过大，导致溢出。而我们只想得到最后取模的结果。
原理 (ab) % c = ( (a%c)(b%c) ) % c

求最大公约数

求解ax + by = gcd(a , b) = d

这里解出来的是特解

求解 ax + by = c

c必须要是 gcd(a,b) 的整数倍才有解
如果要求得到的x是正数，则加上 x1 = (x1%b+b)%b;
