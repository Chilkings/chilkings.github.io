有多个重载函数sqrt实例与参数列表匹配

发表于 2019-03-21 | 分类于 c语言 | | 阅读次数:

在VS2010种遇到下面的错误

有多个重载函数”sqrt”实例与参数列表匹配

经过查找，发现错误原因，做个记录。

sqrt()中应该使用double型或float型

解决方法：

  • 在定义data的时候是int型，可修改为：sqrt(data*1.0)

  • 或者在定义数据的时候直接定义成double型或float型

记一次重装系统后的坑

发表于 2019-03-17 | 分类于 电脑知识 | | 阅读次数:

电脑用着不爽，用前边的傻瓜式重装系统的方法重装了系统。
结果系统装好后，指纹没法用了。
到设备管理器瞄了一眼，发现驱动没装好。第一时间去官网下载了驱动装上。

装好后能正常识别了。
准备设置指纹，结果发现设置按钮是灰的，我内心一万个草泥马。
于是到百度谷歌搜呀搜，什么样的解决方法都有。
最后终于让我发现了问题的原因——win10专业版超级管理员账户是不能录指纹
最后又重装了一次，添加了一个普通账户，并给予管理员权限，再把超级管理员账户禁用了，完美解决。
